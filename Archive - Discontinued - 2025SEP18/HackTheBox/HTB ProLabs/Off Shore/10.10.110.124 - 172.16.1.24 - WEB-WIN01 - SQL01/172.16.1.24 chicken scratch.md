# Burpsuite setup

in `burpsuite pro` navigate to `proxy` tab and ensure that `127.0.0.1:8080` is set as a `Proxy Listener`
![[Pasted image 20231207135120.png]]
navigate to `Proxy Settings -> Network -> Connections` and set proxy host to `127.0.0.1` and `SOCKS proxy port` to `1080` ( whatever port is in `/etc/proxychains4.conf`)
![[Pasted image 20231207135412.png]]
in `firefox` create a `foxy proxy` rule for `burpsuite` where Proxy Type is `HTTP`, Proxy IP address is `127.0.0.1`, and `Port` is 8080
![[Pasted image 20231207135632.png]]
this tells burp to use `8080` for its proxy which is being forwarded through the `socks5` proxy on `1080`
