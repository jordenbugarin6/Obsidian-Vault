##### run wmic service get name, pathname

command run:
`run wmic service get name, pathname` 

notable information:
`pulls all services running on the computer as well as displays the path to the service.`

so what:
`this was utilized to identify that the VulnService1-3 were vulnerable as well as show the paths that they were located at`

notes:
- `wmic service` command is built in windows.
